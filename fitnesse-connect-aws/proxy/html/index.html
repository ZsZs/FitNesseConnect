Fit-Connect

Generic connector library for FitNesse

Fit-Connect is a generic connector library - fixture in FitNesse jargon - to REST services, RDBMS, and Excel files.

This site also demonstrates how I like to use FitNesse, how I like build a test and documentation site. To facilitate a better understanding how the developped connectors works and what the tests are proving, I also build a testbed, a mini application to test.

Enjoy and give me a feedback or report a bug.
