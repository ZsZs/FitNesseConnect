!2 RDBMS Connector
The RDBMS connector has the following fixtures:
 * [[database has record][^DatabaseHasRecord]] - runs a user defined SQL Select and verifies the result set with the given table
 * [[delete record from database][^DeleteRecord]] - deletes records from a table according to the given select statement
 * [[insert record into database][^InsertRecord]] - inserts records into a given table from the the fixture table
 * [[update record in database][^UpdateRecord]] - update records in a table according to the given select statement
