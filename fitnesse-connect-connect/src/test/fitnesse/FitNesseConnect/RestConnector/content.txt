!2 REST Connector - Documentation and Tests
The REST Connector can be used with stateless CRUD fixtures:
 * [[delete resource][^DeleteResource]] - delete resource
 * [[get resource][^GetResource]] - retrives the resource(s)
 * [[patch resource][^PatchResource]] - partially updates the given resource
 * [[post resource][^PostResource]] - creates new resource
 * [[put resource][^PutResource]] - updates given resource
or with statefullscenario fixture:
 * [[REST scenario][^RestScenario]]
